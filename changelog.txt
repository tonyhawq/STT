Version: 0.0.15
Date: 2025-9-23
  Name:
    - The Bugfix Update
  Headline:
    - Fixes a lot of bugs relating to inputs
  Features:
    - User configs are stored in userconfig.json
    - Base configs are stored in internal/config.json
    - If a key is not found in userconfigs, the loader searches in internal/config.json
    - Controls for filters are set in "filter_controls", as a dict of "name": "keybind"
    - To unset a keybind, just set it to "unbound"
  Bugfixes:
    - clicking while submitting say no longer clobbers the chatbar
    - keys are blocked more robustly
    - you no longer randomly walk down after submitting say

Version: 0.0.11
Date: 2025-8-30
  Name:
    - The Group Update
  Headline:
    - Filters can be grouped and disable each other.
  Features:
    - Filters can have groups and be exclusive within that group.
    - If an exclusive filter is activated, all others disable
    - If a non-exclusive filter is activated, any enabled exclusive filters are disabled
    - This only applies to filters within a group
    - A changelog is displayed when you are prompted to update
  Bugfixes:
    - Filters no longer have the chance to increasingly make the window bigger
    - Incorrectly typed values inside of config.json are properly caught inside of config_get_optional_property
